Two ways to get a token — try Option 1 first:

Option 1 — API Key (easiest)
  1. Go to console.anthropic.com → API Keys
  2. Create a new key or use an existing one
  3. Copy and paste it below
Option 2 — Browser Bearer Token
  1. Open claude.ai and log in
  2. Open DevTools: F12 → Network tab
  3. Filter by api/oauth/usage and refresh the page
  4. Copy the Bearer <token> value from Request Headers
  5. Paste it below
